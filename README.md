# 上海火锅数据挖掘

基于大众点评上海749家火锅店，做了一次完整的数据分析实战。

# 数据
- 749条商户记录，含星级、人均价格、口味/环境/服务评分、评论数、推荐菜
- 数据清洗：修正“1元”异常值为111元，评论数中位数填充

# 核心发现
- **价格**：人均100-200元区间的店铺，评论活跃度最高（占35.7%）
- **影响价格的关键因素**：环境评分（相关系数0.74）、口味评分（0.57）
- **商户分群（K-Means，K=3）**：
  - 高端精品店（21家）：人均216元，口味8.9分
  - 大众口碑店（412家）：人均143元，口味8.0分
  - 低价基础店（316家）：人均68元，口味7.2分
- **菜品关联规则（Apriori）**：
  - “毛肚 → 鹅肠”，置信度0.82（吃毛肚的店82%也有鹅肠）
  - “牛肉 → 虾滑”，置信度0.76
- **高频推荐菜 Top 5**：毛肚（89次）、牛肉（76次）、虾滑（62次）、鹅肠（51次）、黄喉（43次）

## 业务建议（基于数据）
- 定价主推150元档位，配套“毛肚+鹅肠”组合套餐
- 装修投入应占总投资30%以上（环境对溢价贡献最大）
