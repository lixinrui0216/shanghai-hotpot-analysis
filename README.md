# 上海火锅餐饮数据挖掘与消费者行为策略分析（双引擎-MySQL与Ptyhon）

## 项目简介
本项目基于大众点评 749 家上海火锅店数据，分别使用 **Python** 和 **MySQL** 两种工具进行数据分析，形成互补的洞察链路。

## 两个项目的特点
| 维度 | Python 项目 | SQL 项目 |
|---|---|---|
| 定位 | 可视化分析 + 算法建模 | 数据清洗 + 基础数据查询 |
| 核心技术 | Pandas、Matplotlib、Sklearn、Jieba | MySQL Workbench、SQL（DDL、DML） |
| 主要输出 | 可视化图表、聚类模型、词云 | 基础查询脚本、业务报表 |
| 适用场景 |可视化与建模分析报告 | 基础数据与指标查询 |

# 数据
- 749条商户记录，含星级、人均价格、口味/环境/服务评分、评论数、推荐菜
- 数据清洗：修正“1元”异常值为111元，评论数中位数填充

# 核心发现
- **价格**：人均100-200元区间的店铺，评论活跃度最高（占35.7%）
- **影响价格的关键因素**：环境评分（相关系数0.74）、口味评分（0.57）
- **商户分群（K-Means，K=3）**：
  - 高端精品店（21家）：人均216元，口味8.9分
  - 大众口碑店（412家）：人均143元，口味8.0分
  - 低价基础店（316家）：人均68元，口味7.2分
- **菜品关联规则（Apriori）**：
  - “毛肚 → 鹅肠”，置信度0.82（吃毛肚的店82%也有鹅肠）
  - “牛肉 → 虾滑”，置信度0.76
- **高频推荐菜 Top 5**：毛肚（89次）、牛肉（76次）、虾滑（62次）、鹅肠（51次）、黄喉（43次）

## 业务建议（基于数据）
- 定价主推150元档位，配套“毛肚+鹅肠”组合套餐
- 装修投入应占总投资30%以上（环境对溢价贡献最大）
